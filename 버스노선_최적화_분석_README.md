# 세종시 버스 노선 최적화 분석 📊

## 개요
세종시의 67개 버스 노선을 대상으로 수요-공급 매칭 분석을 수행하고, 각 노선별 최적화 제안을 제공합니다.

## 주요 최적화 제안

### 1. 배차간격 조정
- **고수요 노선** (50만명 이상): 8-12분 간격 권장
- **중수요 노선** (20-50만명): 15-18분 간격 권장
- **저수요 노선** (20만명 미만): 20-30분 간격 유지

### 2. 운행횟수 증편
- 고수요 노선: 일 100회 이상 운행
- 중수요 노선: 일 80회 운행
- 수요 대비 과잉 운행 노선: 효율성 개선 필요

### 3. 커버리지 확대
- 커버 지역이 1-2개인 노선: 경유지 추가 검토
- 주요 수요지역 미연결 노선: 노선 조정 필요
